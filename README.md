# bids-util

Small Go utility packages for common backend tasks.

## Packages

- `env`: helpers for reading and validating environment-based configuration.
- `logging`: lightweight structured-style logging around application components.
- `passwords`: password hashing and verification utilities with secure defaults.
- `requests`: HTTP request helpers for middleware, auth, CORS, validation, and JSON responses.
- `retries`: simple retry support for operations that may fail temporarily.
- `validation`: reusable field and tag-based validation helpers for request data and user-facing values.
